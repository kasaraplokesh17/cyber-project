# 🔐 Cyber Security Practice Project

This repository contains a collection of Python-based scripts developed as part of my learning journey in cybersecurity. The primary goal of this project is to gain practical knowledge of networking, security concepts, and ethical hacking techniques by implementing them through hands-on coding.

## 🎯 Purpose of the Project

The main objectives of this project are:

- To build a strong foundation in cybersecurity concepts  
- To improve my Python programming skills through practical implementation  
- To understand how common security tools and techniques work internally  
- To gain hands-on experience with networking, scanning, and system interaction  
- To prepare myself for advanced studies and a professional career in cybersecurity  

---

## 🛠️ Topics Covered

This project includes scripts related to:

- Network scanning (ARP-based LAN scanning)  
- Port scanning using different approaches  
- Basic Man-in-the-Middle (MITM) concepts  
- MAC address modification (Windows)  
- Wi-Fi testing and related scripts  
- EXIF data extraction and removal  
- Device discovery and interaction (e.g., smart devices)  
- Basic system-level scripting and automation  

---

## 📂 Project Structure

The repository includes multiple Python scripts, each focusing on a specific cybersecurity concept or task. These scripts are organized for practice and experimentation purposes.
